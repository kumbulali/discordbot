# Discord bot for Minecraft Server
Reward system from Discord for your Minecraft Server.

After inviting the bot to your discord server and typing the following code in the discord channel,
it is a system that can automatically give the desired item to the character's inventory on the connected server.

`!ödül <Nickname> <Server>`

This command can be run every 24 hours.

In order for the Discord bot to work, you need to add the plugin to the server.
